# OryxLab-Pro 윈도우 실행 가이드

OryxLab-Pro 예약 시스템을 윈도우 환경에서 쉽게 실행하기 위한 가이드입니다.

## 1. 사전 준비 (필수)
이 프로젝트를 실행하려면 **Python**이 설치되어 있어야 합니다. (이미 설치되어 있다면 이 단계는 건너뛰세요)

1. [Python 다운로드 페이지](https://www.python.org/downloads/) 접속
2. **Python 3.9** 이상의 버전을 다운로드 및 설치
3. ⚠️ 설치 화면 하단의 **"Add Python to PATH"** 체크박스를 **반드시** 선택해주세요.

## 2. 초기 설정
`setup_env.bat` 파일을 더블 클릭하여 실행하세요.
- 자동으로 가상 환경(`.venv`)을 만들고 필요한 라이브러리를 설치합니다.
- "설정이 완료되었습니다!" 메시지가 나오면 창을 닫아도 됩니다.

## 3. 서버 실행
`run_app.bat` 파일을 더블 클릭하여 실행하세요.
- 검은색 창이 열리고 서버가 시작됩니다.
- 웹 브라우저를 열고 `http://127.0.0.1:5000` 주소로 접속하면 예약 시스템을 사용할 수 있습니다.

## 4. 관리자 로그인
- 주소: `http://127.0.0.1:5000/login`
- 기본 비밀번호: `admin123!`
  - 로그인 후 설정을 변경할 수 있습니다.

## 문제 해결
- **'python'은(는) 내부 또는 외부 명령...** 에러: Python 설치 시 PATH 옵션을 체크했는지 확인하세요. 재설치하거나 PC를 재부팅해보세요.
- **보안 경고**: 윈도우 디펜더 등이 스크립트 실행을 막을 경우 "추가 정보 -> 실행"을 선택하면 됩니다.
